# やったこと
## 設定ファイルを修正

```
sudo vi /etc/rsyslog.d/50-default.conf
```

```
# 該当の行のコメントアウトを外す
# cron.* /var/log/cron.log
↓
cron.* /var/log/cron.log
```

これで/var/log/cron.logにログが出力される
