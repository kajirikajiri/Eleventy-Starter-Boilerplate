# eslint グローバル変数

testcafeを使っていて、test, fixtureという単語でundefinedのErrorが出たので、回避法を調べた

## 回避１

```javascript
/* global test, fixture */
```

## 回避２

```javascript
/* eslint-disable no-undef */
```

回避１だと、test, fixtureがglobalに定義されていることになる

回避２だと、undeinedのErrorがでなくなる
